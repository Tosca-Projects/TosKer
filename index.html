Specify, complete, run!

TosKer is an open source framework for deploying and managing TOSCA applications with Docker. TosKer permits specifying the components forming an application in TOSCA, it then automatically discovers the Docker containers offering the support needed by such components, it include them in the specification, and it then run the application on top of Docker.

The TosKer framework

The TosKer framework is made up of three main components, one for each of the duties TosKer must carry out.

Specify

Describe the components forming a multi-component application with a set of predefined TOSCA types

Complete

Automatically discover the Docker containers satisfying components' hosting requirements with TosKeriser

Run

Run the obtained TOSCA application specifications on top of Docker, with the TosKer engine
